Linux 版本打包脚本（file_change）

脚本简介

该脚本用于打包 Go + Fyne 开发的 file_change 应用，适用于 Linux 系统。脚本包含依赖安装、源码编译、图标生成、Fyne 打包等完整流程，具备容错机制，确保即使 Fyne 打包失败也能提供手动编译产物。

核心配置

配置项 说明 取值
PROJECT_DIR 项目根目录 $HOME/GO_Pro/file_change
APP_NAME 应用名称 file_change
APP_VERSION 应用版本 1.0.0
APP_BUILD 构建版本 1
OUTPUT_DIR 输出目录 $PROJECT_DIR/dist/linux
TEMP_ICON 临时图标路径 $PROJECT_DIR/Icon.png
GO_BIN_PATH Go 二进制文件路径 /usr/local/go/bin

动态路径

  • GOPATH：从环境变量获取，取第一个路径
  • FYNE_CMD：Fyne 工具路径（$GOPATH/bin/fyne）
  • EXECUTABLE_PATH：手动编译输出路径（$PROJECT_DIR/$APP_NAME）

注意事项

  1. 脚本使用 dnf 包管理器，适用于 Fedora、RHEL 等系统；若使用 Ubuntu/Debian，需将 dnf 替换为 apt-get，并调整依赖包名称
  2. 确保已安装 Go 环境（版本 ≥ 1.16）
  3. 若已安装 Fyne 工具，脚本会直接使用；若未安装，需先执行 go install fyne.io/fyne/v2/cmd/fyne@latest
  4. 脚本具备幂等性，可多次运行，不会产生冗余文件
